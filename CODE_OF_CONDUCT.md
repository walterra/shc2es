# Code of Conduct

## About This Project

This is an early-stage project maintained by a single person. While the project is small, these guidelines help set expectations for respectful interaction.

## Expected Behavior

- Be respectful and constructive in discussions
- Give and accept feedback gracefully
- Focus on the technical merits of contributions
- Respect differing viewpoints and experiences

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information without permission

## Reporting Issues

If you experience unacceptable behavior, please contact the maintainer via email (see `package.json` author field).

## Enforcement

The maintainer reserves the right to remove comments, reject contributions, or block users who violate this code of conduct.
