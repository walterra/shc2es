Bosch Smart Home nodejs based project to collect device data via long polling. Data to be stored as NDJSON and to be visualized as time series.

## Hardware

This project is designed to work with the [Bosch Smart Home Controller II](https://www.bosch-smarthome.com/at/de/produkte/steuerung-und-zentrale/smart-home-controller/).

### Controller II Features

- **Communication**: ZigBee 3.0 (2.4 GHz), Matter-ready as bridge device
- **Connectivity**: RJ45 Ethernet (10/100 Mbit/s)
- **Security**: Local encrypted data storage, secure remote access
- **Local API**: Supports private/non-commercial developer solutions (documented on GitHub)
- **Integrations**: Amazon Alexa, Apple HomeKit, Google Assistant
- **Supported Devices**: 2nd-gen Bosch Smart Home products; 1st-gen devices via optional 868 MHz Funk-Stick
